<h3 align="center">PERN Stack App</h4>

- React (Redux)
- Node.js (Express)
- PostgreSQL
- nginx
- Docker

## Installation

1)  
    ```bash
    git clone https://github.com/raevsanton/bookstore.git
    ```
2. 
   ```bash
    cd bookstore
    ```
3.
    ```bash
    docker-compose up --build
    ```
4.  
    ```bash
    http://localhost:8080
    ```
